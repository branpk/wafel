Abstractions for reading and writing values to game memory.

This crate defines the MemoryRead and MemoryWrite traits which can be used to read/write Values to arbitrary addresses.

It also defines the GameMemory trait which distinguishes between static and non-static memory and provides state saving/loading functionality.

Finally it provides DllGameMemory which implements GameMemory using a game DLL, and EmuMemory which attaches to a running emulator.
