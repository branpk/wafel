Expand description

A trait with methods for reading from game memory.

This needs to be implemented so that interpret_f3d_display_list can read objects from memory (viewports, textures, display lists, etc).

Required Associated Types

source

type Ptr: Debug + Copy + PartialEq

The pointer type that can be read from a display list command.

PartialEq is only used for checking if the color image is equal to the depth image.
