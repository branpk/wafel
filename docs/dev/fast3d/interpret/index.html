Module fast3d::interpret

source · [−]
Expand description

A display list interpreter that produces a self-contained F3DRenderData object that is straightforward to render.

After implementing F3DMemory for loading vertex/texture/dls/etc data from memory, interpret_f3d_display_list can be called.

Note: this module is not complete and may have errors.
