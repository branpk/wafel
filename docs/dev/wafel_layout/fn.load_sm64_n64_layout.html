Return the layout for SM64 on the N64, including data added by DataLayout::add_sm64_extras.

version can be “us”, “jp”, “eu”, or “sh”.

for version in ["us", "jp", "eu", "sh"] {
    wafel_layout::load_sm64_n64_layout(version).unwrap();
}
