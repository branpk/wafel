fn apply(&self, memory: &M, slot: &mut M::Slot, frame: u32) -> Vec<Self::Error>

Apply edits to the given state.

Even if this method returns an error, the edits that were made to slot are still incorporated into the timeline. The errors can be queried from the timeline.

This method must be deterministic.
