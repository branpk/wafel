Expand description

Find a raw josytick value that approximately maps to the given intended inputs.

If the given input has maximum magnitude, then try to produce a raw input in a nearby HAU of the intended yaw (relative to relative_to). If it does not have maximum magnitude, then return a raw joystick that maps to a nearby adjusted input, without worrying about exact angle or magnitude.
